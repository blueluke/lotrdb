About

The Rivendell Councilroom is a deck builder for the Lord of the Rings LCG by Fantasy Flight Games.

It was built from scratch using AngularJS.

The Rivendell Councilroom was created by Dirk Meijer and published under the GNU GPL (version 2) in April 2015.

The source code is available on the project page.
Please refer to the Github project page for more information on the development of the Rivendell Councilroom.
